按[测评网站](http://assess.tiger-code.com/)的测评方法，常用6000字，各单字输入方案，重码率如下：

```
虎码 (0.01%) > 
五笔 (0.14%) > 
自然虎形 (0.38~0.54%) > 
小鹤音形 (1.01%~1.38%) > 
自然码 (2%~2.95%) > 
纯双拼 (45%~52%)
```

虎码、五笔的数据，直接取自测评网站。

其他四种，用 “8105单字码表” 目录下的文件，上传到测评网站，所得到的数据。
