按[测评网站](http://assess.tiger-code.com/)的测评方法，常用6000字，各单字输入方案，加权统计的重码率如下：

```
虎码 (0.01%) < 五笔 (0.14%) < 
自然虎形 (0.38%~0.54%) < 自然表形 (0.79%~1.02%) < 小鹤音形 (1.01%~1.38%) < 自然码 (2%~2.95%) < 
纯双拼 (45%~52%)
```

虎码、五笔的数据，直接取自测评网站。

其他四种，用 “8105单字码表” 目录下的文件，上传到测评网站，所得到的数据。

BTW, 如果会“表形码”，也可用 双拼 + 表形首末。可参考这个 [小鹤表形](https://tieba.baidu.com/p/8528130134)，它的数据似乎也很好。
