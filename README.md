# 汉字输入法

## 汉字

最新的国际标准 Unicode 15.1，从 CJK 到增补的 CJK-A ~ CJK-I，共有约 97681 汉字，涵盖东亚的汉字圈各地区的古今用字。

除非从事古代汉语相关的工作，近 10 万汉字中的绝大部分汉字，一般人都不会用到。就算是《四库全书》，也用不到其中的很多字。

大部分人，三四千汉字已满足现代汉语日常交流的 99.9% 以上的使用。（日本现在大约使用两千日文汉字）

## 拼音

[拼音pdf](https://github.com/impishian/input_method/blob/main/tex/pinyin.pdf) ，根据大陆学校教育人手一本的《新华字典》（第12版），整理列出共416种声韵组合。除了6种语气词的读音，其他 410 种，一般的拼音输入法都支持。而那6种，所对应的8个语气词是多音字，用其他的声韵组合也能打出。

通常，拼音输入法，用26键（无声调）打字，重码率高。大陆学过拼音的人，基本不用学习就能上手打字。

## 注音

[注音pdf](https://github.com/impishian/input_method/blob/main/tex/pinyin_zhuyin.pdf) ，整理列出了 407 种声介韵组合，基本与拼音的组合一致。

注音输入法，通常需要用40键来打字。注音带声调，多打声调键，重码就少一些。

注音打字，击键比拼音少，比双拼多，不如双拼的打字节奏稳定。台湾学过注音的人，了解了40键和注音符号的对应关系，就可打字。

大陆的《新华字典》，每个字除了标了拼音，也标着注音，几十年来多个版本修订，到现在的最新的第 12 版，依旧保留注音。注音符号本来就是源自汉字的偏旁部首。大陆人稍加学习，也能掌握注音。

## 郑码（郑易里、郑珑父女）

[中易官网郑码介绍](https://www.china-e.com.cn/li/main/zhengma/jj.htm)

纯形码，字根双编码，约 186 字根。不用口诀背字根，理解笔画规律，先记 52 个主根，再记其余的 134 个副根。

从一推出时，就是面向大字符集（数以万计的字的集合）的输入法。不像五笔那样最初只面对GB2312的六千多字来设计字根，后来还推出98版、新世纪版等多个版本。

若以大陆字型为准，郑码与五笔相比，编码更规范。拆字更舒服。

不满意郑码者，是郑码规则导致的“简全不一致”（简码和词组编码跟全码不同，简码并不是全码前几键）

受郑码字根双编码的思路影响，近些年出现了一些字根很多的字根双编码的输入方案。

## 五笔（王永民）

[王码官网](http://www.wangma.net.cn/)

**五笔是商业推广最成功的形码**。

86版，使用最广（后有多个版本，坊间魔改的五笔也很多），约130字根。字根单编码，通常用口诀背诵记忆字根。

86版，最初主要面向 GB2312 的 6763 个字。后来，逐渐用于大字符集，可能重码会过多。

末笔交叉识别码，会有些人不适应。

另外，有些字拆字别扭。（当然，汉字几千年的演变，非常复杂，没有一种形码，拆字会绝对舒服，绝对规范。况且也没绝对的《规范》可供多国家多地区的所有汉字使用者遵守）

另有说法，86版使用234个字根，98版使用259个字根，新世纪版使用了226个字根。应该是形似根不归并，这样统计而得的数量。

五笔早期商业推广最成功，至今也是使用者最多的形码，一些爱好者还魔改了各种各样的 五笔 方案，追求极致输入体验。

## 仓颉（朱邦復）

[朱邦復工作室](http://www.cbflabs.com/)

与五笔、郑码等用字根拆字组字的方式不同，仓颉把字看成一个图像，类似四角号码查字法那样，给字的各个角取码打字。

五笔、郑码等形码最多四码。仓颉最多时会用五码打一字。用最多五码来编码，好处是重码少。另外，仓颉不设简码。

仓颉有第三代、第五代等，差别不大。还有只取首尾仓颉码的 速成输入法（或称简易输入法）。

第五代手册：http://cbflabs.com/book/ocj5/ocj5/index.html

苍颉，是坊间所称的第六代，但与常见的三代五代差异很大。更像是衍生的输入法。

## 二笔输入法（陈劲松）

曾通过中国教育部评审，可以进入中小学教材。2013年专利期已满。现有一些输入方案，是基于原二笔修改而实现的方案。

## 自然码（双拼+形）（周志农）

[自然码官网](http://www.ziranma.com.cn/)

自然码是双拼的鼻祖之一。双拼，声韵各一键，打字节奏稳定，并可减少击键。

2009年3月26日更新到7.27版本后，停止了更新。不少爱好者在主流输入法中定制实现自然码。

自然码辅助码，可减少同音字的重码。（有双辅方案、单辅方案，即：双码、单码的辅码。）

## 虎码 (PeaceB)

[虎码官网](https://tiger-code.com/)

(1) 字根

虎码 241 字根，把五笔、郑码的主要字根绝大部分都收了进来，再加了几十个字根，因此，字根数比五笔、郑码都多不少。

官网提供工具来辅助记忆，也不难记住字根。（像打游戏一样，练习超过3万分，则字根基本过关）

虎码是按特定的计算机算法，将各字根安排在特定的键，不像五笔或郑码那样人为按一定的规律排布。字根完全乱序。

虎码像郑码一样采用双编码，一个字根，有大码、小码（小码音托。不懂拼音者可另想办法记小码，问题也不大）。

不要像五笔那样背字根，可用 241 个词语来记（比如：通用的 "用" Ty，方且的 "且" Fq，誓言的 "言" Sy，地里的 "里" Dl ...）

虎码面向大字符集。字根多，打字体验就比较好。像搭积木一样，能轻易搭出各种字。比五笔更容易拆字、重码少、极少的无理码。

(2) 规则

**虎码，是取码规则最简单的形码**

五笔、郑码等，需要看书或通过打字培训班来学习掌握复杂的规则、特例，甚至还要记无理码。 

虎码规则大致如下（ ① 大写字母：大码；② 小写字母：小码；③ 2表示取两根：Aa 或 AB）：

|     | 一 | 二 | 三 | 多 | 备注 |
|------|------|------|------|------|------|
| 字 | 一根<br>Aa |二根<br>ABb|三根<br>ABCc|多根<br>ABCZ| 一句话：打各字根大码，不够4码补最后字根小码 |
| 词 | | 双字<br>2+2|三字<br>A+A+2|多字<br>A+A+A+...+A|  |

虎码的简洁的官网里有关于虎码的所有内容，不需另外看书或跟老师学。 

(3) [虎码测评网](http://assess.tiger-code.com/)

这是虎码网站上提供的一个测评工具。

任何输入方案的码表（必要时转换一下格式），上传后，可测评该输入方案在常用6000字集的表现情况：

① 离散情况：用1/2/3/4码能打出的字，分别占比有多少，静态重码率多少；

② 效率：理论二简，加权键长；

③ 手感： 左右互击、手指跨排、 同指/同键连打 ... 等。

## 顶功（戴石麟）

戴石麟是“顶功”理论提出者。

通过巧妙的预设规则，减少空格的使用，缩短码长，自动上屏，实现更高效输入。

但是，顶功需要学习和记忆的东西多。

## 并击

并行击键，高速输入，主要用于速录机的领域。
